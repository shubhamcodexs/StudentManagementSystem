# 📚 Student Record Management System (Java CLI)

## 🎯 Objective
This project is a **Command Line Interface (CLI)-based CRUD application** developed in Java to manage student records. It allows users to perform basic operations like adding, viewing, updating, and deleting student data.

---

## 🛠️ Tools & Technologies
- Java
- VS Code / IntelliJ IDEA Community Edition
- ArrayList (Java Collections Framework)

---

## ✨ Features
- ➕ Add new student record  
- 📄 View all student records  
- ✏️ Update existing student details  
- ❌ Delete student record  
- 🔁 Menu-driven interface
